01985 !
01986 ! !DESCRIPTION:
01987 !
01988 ! Calculates the sin of latitudes on the grid based on ny_global and
01989 ! using code from CAM (/control/gauaw_mod.F90)  In CAM, gauaw_mod.F90 is
01990 ! only used to calculate the sin of latitudes (and thence latitude) if
01991 ! the dynamical core is set to eul.  If using one of the other dynamical
01992 ! cores and coupling to stand alone CAM the latitudes calculated in this

02005

02015            sinlats      ! sine of latitudes
02016 
02017       real (kind=dbl_kind) :: 
02018            eps      ,  ! convergence criterion
02019            c        ,  ! constant combination
