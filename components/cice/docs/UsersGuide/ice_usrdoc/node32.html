Changing Content

The second namelist in the setup script controls what variables are written to the history file. To remove a field from this list, add the name of the character variable associated with that field to the &icefields_nml namelist in cice.buildnml.csh and assign it a value of 'xxxxx'. For example, to remove ice thickness and snow cover from the history file, add 

&icefields_nml
    f_hi   =  'xxxxx'
  , f_hs   =  'xxxxx'
/
to the namelist.
