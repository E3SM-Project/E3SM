Thermodynamic Iteration Error

This error is written from ice_therm_vertical.F90 when the ice model temperature iteration is not converging in the thermodynamics. This is usually a problem with the forcing, but sometimes can be indicative of a timestep problem in the ice. 

  Thermo iteration does not converge
  istep1, my_task, i, j:
